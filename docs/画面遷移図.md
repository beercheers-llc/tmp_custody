# 画面遷移図

本ドキュメントでは、要件定義書と業務フロー図に基づいた画面遷移図を示します。

## 1. ログイン・ユーザー管理系

```mermaid
flowchart TD
    A[ログイン画面] -->|認証成功| B[ダッシュボード]
    A -->|認証失敗| A
    A -->|パスワード忘れ| C[パスワードリセット]
    C -->|メール送信| A
    B -->|ユーザー管理| D[ユーザー一覧]
    D -->|ユーザー追加| E[ユーザー登録]
    D -->|ユーザー編集| F[ユーザー情報編集]
    D -->|権限変更| G[権限設定]
```
